Challenges and Lessons Learned

One major challenge was maintaining consistent layout and style across all pages while using the shared template. I also had to carefully manage positioning when layering video and text on the Splash Screen. From these tasks, I learned how to apply z-index, flexbox, and accessibility principles like semantic HTML, contrast handling, and alt descriptions.

Compliance

All of my web pages have been validated using the W3C HTML Validator. I ensured proper nesting of elements, semantic HTML5 usage, and no critical validation errors. Additionally, I complied with the JISC Accessibility Regulations by including alt text for all images, using accessible labels, and ensuring keyboard navigation where relevant.
